《恶意》读书笔记

七月 03, 2019
  • 东野圭吾
  • 悬疑推理
  • 读书笔记

无边的恶意，深不见底，有如万丈深渊。

杀人不是目的，而是手段；死亡不是结束，而是开始。

很久没有读过书了，生活似乎进入了恶性循环，做不完的工作，睡不醒的觉。一切都与自己最初的想法背道而驰，完全没有动力，对生活没有新鲜感，这样的状态让我恐慌。为了抑制那点恐慌，最近拿起了很久没摸过的书，发现还有几本未曾开封，挑了一本随便读起来。东野圭吾的文字还是那么的吸引人，所以没有停下来，花了两天时间读完了这本《恶意》。
读完整本书，感觉东野对整个故事情节的把握做到了极致，情节的不断反转令人感叹，这是东野的特点。除了对东野在推理小说上的精湛技艺表示感慨，他对人性的思考也值得我们深思。当一个人为了某个目的，采取杀人这种手段，展示出来的人性让我惊恐，但是在这个社会上何尝不是这样呢。东野他是对人性、对社会做过多么深入的思考呀。
真的是：这世间唯太阳和人心不能直视。
